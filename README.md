# spy-fall
Creating a version of the Spyfall game using TypeScript, React, and Socket.io

This project is not complete and is not stable.

## Libraries

- TypeScript
- React
- Reach Router
- Webpack
- Socket.io

## Run

Server:

    $ cd server
    $ npm install
    $ npx tsc
    $ node dist/index.js

Client:

    $ cd client
    $ npm install
    $ npm start
